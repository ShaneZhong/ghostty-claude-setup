# Yazi Keyboard Shortcuts

Reference: https://kb.adamsdesk.com/application/yazi-keyboard-shortcuts/

## Navigation

| Shortcut | Action |
|----------|--------|
| `h` / `Left` | Parent directory |
| `l` / `Right` | Enter directory |
| `j` / `Down` | Move down |
| `k` / `Up` | Move up |
| `G` | Move to bottom |
| `g+g` | Move to top |
| `Ctrl+d` | Half-page down |
| `Ctrl+u` | Half-page up |
| `Ctrl+f` | Full page down |
| `Ctrl+b` | Full page up |
| `J` / `K` | Scroll preview down/up (5 units) |
| `g+h` | Go to home directory |
| `g+d` | Go to downloads directory |
| `g+c` | Go to config directory |
| `g+Space` | Interactive directory change |
| `z` | Change directory with zoxide |
| `Z` | Reveal with fzf |

## File Operations

| Shortcut | Action |
|----------|--------|
| `Enter` / `o` | Open file |
| `O` / `Shift+Enter` | Open interactively |
| `y` | Copy selected files |
| `x` | Cut selected files |
| `p` | Paste files |
| `P` | Paste (overwrite existing) |
| `d` | Delete to trash |
| `D` | Permanently delete |
| `r` | Rename |
| `a` | Create file (end with `/` for directory) |
| `X` / `Y` | Cancel cut/copy |
| `-` | Symlink (absolute path) |
| `_` | Symlink (relative path) |

## Copy Paths

| Shortcut | Action |
|----------|--------|
| `c+d` | Copy directory path |
| `c+c` | Copy file path |
| `c+f` | Copy filename |
| `c+n` | Copy filename without extension |

## Selection

| Shortcut | Action |
|----------|--------|
| `Space` | Toggle selection |
| `Ctrl+a` | Select all |
| `Ctrl+r` | Inverse selection |
| `v` | Enable visual mode |
| `V` | Disable visual mode |
| `Esc` | Cancel selection |

## Search & Filter

| Shortcut | Action |
|----------|--------|
| `s` | Search by name (fd) |
| `S` | Search by content (ripgrep) |
| `f` | Filter files |
| `/` | Find next file |
| `?` | Find previous file |
| `n` / `N` | Navigate found items |
| `Ctrl+s` | Cancel search |

## Tab Management

| Shortcut | Action |
|----------|--------|
| `t` | New tab |
| `1`~`9` | Switch to tab N |
| `[` / `]` | Previous/next tab |
| `{` / `}` | Swap with previous/next tab |
| `Ctrl+c` | Close current tab (quits if last) |

## Sort

| Shortcut | Action |
|----------|--------|
| `,+A` | Alphabetically |
| `,+n` / `,+N` | Natural / reverse |
| `,+m` / `,+M` | Modified time / reverse |
| `,+c` / `,+C` | Creation time / reverse |
| `,+e` / `,+E` | Extension / reverse |
| `,+s` / `,+S` | Size / reverse |
| `,+r` | Random |

## View

| Shortcut | Action |
|----------|--------|
| `.` | Toggle hidden files |
| `w` | Show task manager |

## General

| Shortcut | Action |
|----------|--------|
| `~` / `F1` | Help screen |
| `q` | Quit |
| `Q` | Quit without writing cwd |
| `Ctrl+z` | Suspend process |
| `;` / `:` | Run shell command |
