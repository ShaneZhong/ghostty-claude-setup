# Ghostty macOS Keyboard Shortcuts

## Splits

| Shortcut | Action |
|----------|--------|
| `Cmd+D` | Split right |
| `Cmd+Shift+D` | Split down |
| `Cmd+[` | Previous split |
| `Cmd+]` | Next split |
| `Cmd+Option+Up/Down/Left/Right` | Navigate to split in direction |
| `Cmd+Ctrl+Up/Down/Left/Right` | Resize split (10 cells) |
| `Cmd+Ctrl+=` | Equalize all splits |
| `Cmd+Shift+Enter` | Toggle split zoom (fullscreen/restore) |
| `Cmd+W` | Close focused split |

## Tabs

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+W` | Close current surface |
| `Cmd+Shift+W` | Close window (all tabs) |
| `Cmd+1~8` | Jump to tab N |
| `Cmd+9` | Jump to last tab |
| `Cmd+Shift+[` | Previous tab |
| `Cmd+Shift+]` | Next tab |
| `Cmd+Shift+T` | Undo close tab |
| `Cmd+Shift+R` | Rename tab (custom keybind) |
| `Cmd+Shift+R` | Rename tab (custom keybind) |

## Windows

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New window |
| `` Cmd+` `` | Cycle windows (macOS system) |
| `Cmd+Enter` | Toggle fullscreen |
| `Cmd+Q` | Quit |

## Clipboard

| Shortcut | Action |
|----------|--------|
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Cmd+Shift+V` | Paste from selection |
| `Cmd+A` | Select all |

## Font

| Shortcut | Action |
|----------|--------|
| `Cmd+=` | Zoom in |
| `Cmd+-` | Zoom out |
| `Cmd+0` | Reset font size |

## Scrolling

| Shortcut | Action |
|----------|--------|
| `Cmd+Home` | Scroll to top |
| `Cmd+End` | Scroll to bottom |
| `Cmd+PageUp/PageDown` | Scroll page up/down |
| `Cmd+Up/Down` | Jump to previous/next prompt (requires shell integration) |

## Search

| Shortcut | Action |
|----------|--------|
| `Cmd+F` | Start search |
| `Cmd+G` | Next match |
| `Cmd+Shift+G` | Previous match |
| `Escape` | End search |

## General

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Clear screen |
| `Cmd+Shift+,` | Reload config |
| `Cmd+Shift+P` | Command palette |
| `Cmd+Option+I` | Toggle inspector |
| `` Ctrl+` `` | Quick Terminal (global, from any app) |

## Natural Text Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+Left` | Beginning of line |
| `Cmd+Right` | End of line |
| `Cmd+Backspace` | Kill line backward |
| `Option+Left/Right` | Move word backward/forward |
